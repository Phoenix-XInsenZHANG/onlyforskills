# D11 Rules

## Collection Names
D11 collections are **singular**: `order` not `orders`, `order_line` not `order_lines`.

## D11_DEV Hard Gate
If `D11_DEV` is not set in `.env.local`, **stop and ask** before any D11 data work.

## Authenticated Fetch
Use `authenticatedFetch` (not axios, not raw fetch) for D11 API calls. Auto-handles 401.

## Self-Guard Pattern
Every D11-only function must self-guard:
```typescript
if (process.env.NEXT_PUBLIC_AUTH_POLICY !== 'true') return
```
This prevents D11 calls from firing in D9 auth modes.

## Org Identity

Use `Number(org.code)` (stable business ID, e.g. `63`), NOT `org.id` (auto-increment PK).

## RBAC Chain

`directus_users → directus_access → directus_policies → directus_permissions`
- Policies are **additive** (union/OR)
- Frontend should **not** pass `orq` — D11 presets + AND filter handle scoping server-side
